Es un archivo .war que se puede levantar en cualquier servidor tomcat o glasfish.

una API REST llamada : url/fechas/detalle

JSON formato parametros de entrada (body):

ejemplo:
{
    "id": 6,
    "fechaCreacion": "1969-03-01",
    "fechaFin": "1970-01-01",
    "fechas": [
      "1969-03-01",
      "1969-05-01",
      "1969-09-01",
      "1970-01-01"]
}


Salida:

ejemplo nivel 3
